std::tuple

std::tuple in <tuple> is a container for containing N items, where N and the type of each item is fixed at compile-time. You use std::get to access that item. The index must be known at compile-time.

#include <tuple>
#include <iostream>
int main() {
    std::tuple<int, int, int> x;
    std::get<0>(x) = 5; // Sets the first value to 5.
    std::cout << std::get<0>(x); // Prints first value.
}

You can have a tuple of as many elements as you like. Tuples are copyable, movable, and comparable if their element types are. The first notable support function is std::tie. The second is std::make_tuple.

#include <tuple>
int main() {
    int a, b, c;
    std::tie(a, b, c) = std::make_tuple(0,0,0);
}

std::tie creates a tuple of references to it's arguments. std::make_tuple creates a tuple of it's arguments. When they are assigned, each element in the right-hand-side tuple is assigned to an element in the left-hand-tuple- and as those elements are references, in reality the tie arguments are modified.

We also meet std::pair in <utility>. It's the baby version of std::tuple, for just two elements. You can access the first element through the first member and the second through the second member. It rarely sees use, as it's almost entirely strictly superseded by std::tuple except that some of the other Standard containers use it in their interface. As with std::tuple, there is a std::make_pair.
