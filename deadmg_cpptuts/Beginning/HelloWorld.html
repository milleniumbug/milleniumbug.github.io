Hello, World

The traditional opening program in any language is Hello, World. It prints the titular text. So without further ado,

#include <iostream>
int main() {
    std::cout << "Hello, World!";
}

However, depending on your environment, you may have run into a curious problem. The above program causes "Hello, World!" to be printed. But it doesn't cause the place where you printed it to continue to exist. If you execute this program in Visual Studio, and some other development environments, the console window will disappear before you could possibly have read the output. However, the problem of pausing a program is far more general than this. And so we introduce something called a "breakpoint". A breakpoint instructs an external tool, the debugger, to pause the program so that we can observe, and sometimes even modify, its state whilst executing. The exact sequence is specific to each debugging tool.

Visual Studio: Add, after the semicolon, __debugbreak();. This will cause the debugger to pause your program.

Clang/G++: These compilers execute on the command line (generally) and don't require this in this case. You can wait to look up breakpoints.

Online: These compilers return the whole output as a web page, which doesn't have this problem. But then, they don't have debuggers, so I hope nothing ever goes wrong.

You can have fun changing the output text to whatever you want.

Comments

In addition, we will introduce comments in this chapter. Comments are C++ source code that does not have any executable function and is not compiled. The compiler strips all comments before performing any other action on the source code. Thus, the author of the code leaves instructions or information about the surrounding code in them. In addition, they are used to remove code from the compiler which you may want to re-introduce easily.

#include <iostream>
int main() {

Single-line comments are introduced by a double forward slash. The comment lasts until the next line. You can enter any text in here.

    // This line does nothing. ¬!"£$%^&*()_+{}~@:

Multi-line comments are introduced by a forward slash and an asterisk. They are ended by an asterisk and a foward slash. Strictly, there is no need for them to be multi-line- they can be of arbitrary size and position.

   std::cout << /*"Commented out"*/ "Not commented out";
}
