Input

Taking input from the outside world is an essential process for any program. Nobody would get anything done if they couldn't take input. Unfortunately, it's also a risky business. That user typing on his keyboard doesn't have to enter a number where you think there should be one. He could enter anything. Maybe he just doesn't know what he's supposed to do- or maybe he's trying to attack your program. This means that well-written programs which take input from untrusted sources must always check the correctness of that input. Failure to validate input is a serious problem.

In C++, the Standard iostream library provides an companion object to std::cout. This object is, appropriately, named std::cin. This can be used to take input from the user. The object- and the return value of std::cin >> var- can be implicitly converted to boolean. If the boolean is false, then the input failed. If it is true, the input succeeded. This simple program requests input from the user until it's valid.

#include <iostream>
int main() {
    int i = 0;
    // always best to set a value just in case
    // since if it failed and we read it by accident, UB.
    std::cout << "Please enter an integer.\n";
    // While the user did not enter a valid integer
    while(!(std::cin >> i)) {
        // Ask the user for new input
        std::cout << "Could not determine value from input. Please enter again.\n";
        // Tell std::cin that it's OK to try again
        std::cin.clear();
        /* Tell cin to forget everything up until the next line of user input
        so we don't repeatedly try to read the same bad input. */
        std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
    }
    std::cout << "The integer you entered was " << i << "\n";
}

Like std::cout, std::cin can deal with a variety of data types, including strings, integers, and more. std::cin and std::cout both have the curious property of returning... std::cin and std::cout. This means that

std::cout << "Oh" << "ai";

is perfectly legal and valid C++, and is quite equivalent to

std::cout << "Oh";
std::cout << "ai";
